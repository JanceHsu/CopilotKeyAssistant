# CopilotKeyAssistant

### 使用 PowerToys 将你的 Copilot 快捷键映射到自定义 AI Portal 。

#### 操作步骤

1. 在 Microsoft Store 中安装 PowerToys 工具，并打开开机自启动。
2. 下载 AI Portal ，并放置在任意路径下，注意不可以随意变更该路径。
3. 打开 PowerToys ，在键盘管理器中，选择**重新映射快捷键**，选择**添加快捷方式重新映射**。
4. 选择**铅笔图标**，按下 **Copilot** 键（等价于 `Win (left)` + `Shift (left)` + `F23` ）；操作选择**打开 URI** ，路径填写存储 AI Portal 的 `index.html` 的路径。
5. 更多操作说明，参考 AI Portal 内部使用说明。

**重要：安装好 AI Portal 后，文件路径不可随意改变。路径若改变，需要在 PowerToys 中重新映射。**
