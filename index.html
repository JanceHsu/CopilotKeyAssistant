💡 本地工具启动配置
本地工具一键打开仅支持 Windows 系统。
点击本地工具卡片将通过 launchapp:// 协议直接打开应用程序。首次使用前需要注册协议。
⚡ 快速指引：
• 方法一：点击“下载注册脚本”，双击运行 .reg 文件。若被 Windows 安全中心拦截，请右键 → 属性 → 勾选“解除锁定”。
• 方法二：点击“复制终端命令”，打开 PowerShell（Win + R 输入 powershell），粘贴并回车运行。
• 清除：点击“清除注册协议”下载移除脚本，按照同样的方式运行该脚本。
